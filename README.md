**Time Table Generator** – Ваш помощник в создании школьного расписания!

**Что это такое?**
Программа для быстрого и удобного составления еженедельного расписания занятий. Автоматизируйте процесс распределения уроков и учителей по дням и периодам.

**Возможности:**
- Выбор дня недели (понедельник-пятница)
- Выбор учебного периода (1-7 уроки)
- Назначение предметов и учителей
- Наглядное отображение расписания в таблице
- Мгновенное обновление данных

**Для кого подойдет:**
- Для администраторов школ и учебных заведений
- Для учителей, составляющих расписание
- Для студентов, изучающих Java и GUI-программирование

**Преимущества:**
- Интуитивно понятный интерфейс
- Быстрое назначение уроков
- Компактное отображение всей недели
- Простота использования

Попробуйте прямо сейчас и создайте идеальное расписание за минуту!
